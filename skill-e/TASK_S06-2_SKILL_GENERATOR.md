#### 2. Skill Generation (`generateSkill()`)
```typescript
const result = await generateSkill(optimizedContext, {
  provider: 'anthropic',
  model: 'claude-3-5-sonnet-20241022',
  apiKey: 'your-api-key',
```

**Returns:**
- `markdown` - Complete SKILL.md content
- `frontmatter` - Parsed YAML frontmatter
- `toolDefinition` - OpenAI/Anthropic compatible tool definition (FR-6.17)
- `tokenCount` - Estimated token count
- `metadata` - Generation metadata (provider, model, time)

#### 6. SKILL.md Format
Generates skills following AgentSkills.io specification:

```yaml
---
name: skill-name-here
description: Clear description of what this skill does and when to use it.
compatibility: Desktop application
license: Apache-2.0
metadata:
  author: Skill-E
  version: "1.0"
  recorded: "2024-01-26"
  source: demonstration
---
```

```markdown
# Skill Title

Brief description from narration.

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `{email}` | email | yes | User email address |
| `{password}` | password | yes | User password |

## When to Use

- Situation 1 when this skill applies
- Situation 2 when this skill applies

## Prerequisites

- Required access or permissions
- Required applications open

### Step 1: Navigate to Signup Page

1. Click the "Sign Up" button

> **Note**: This is usually in the top right corner

1. Enter `{email}` in the Email field

**If `{condition}` = `value1`:**
1. Do this path

**If `{condition}` = `value2`:**
1. Do alternative path

## Verification

- [ ] Checkpoint 1 completed
- [ ] Checkpoint 2 completed

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Error X | Try solution Y |

---
*Generated by Skill-E from demonstration recording*
```

### API Integration Details

#### Anthropic API
- Endpoint: `https://api.anthropic.com/v1/messages`
- Headers: `x-api-key`, `anthropic-version: 2023-06-01`
- Supports streaming with Server-Sent Events (SSE)

#### OpenAI API
- Endpoint: `https://api.openai.com/v1/chat/completions`
- Headers: `Authorization: Bearer {apiKey}`
- Standard chat completions format

#### OpenRouter API
- Endpoint: `https://openrouter.ai/api/v1/chat/completions`
- Headers: `Authorization`, `HTTP-Referer`, `X-Title`
- OpenAI-compatible format

## Usage Example

```typescript
import { generateSkill } from './lib/skill-generator';
import { optimizeContext } from './lib/context-optimizer';

// Step 1: Optimize the processed session
const optimizedContext = await optimizeContext(processedSession);

// Step 2: Generate the skill
const result = await generateSkill(optimizedContext, {
  provider: 'anthropic',
  model: 'claude-3-5-sonnet-20241022',
  apiKey: settings.claudeApiKey,
  stream: true,
  onChunk: (chunk) => {
    // Update UI with progressive generation
    console.log('Received chunk:', chunk);
  },
});

// Step 3: Use the generated skill
console.log('Generated SKILL.md:', result.markdown);
console.log('Token count:', result.tokenCount);
console.log('Tool definition:', result.toolDefinition);

// Step 4: Save to file
await saveSkillToFile(result.markdown, 'my-skill');
await saveToolDefinition(result.toolDefinition, 'my-skill');
```
