# Skill-E v1.0.0 🧠

> **The Bridge Between Human Action and Agent Understanding.**

**Skill-E** is a local-first application that observes your workflow and automatically generates structured **Skill Definitions** for AI Agents.

Instead of writing complex prompts or code, you simply *show* the agent what to do while explaining it with your voice.

### 🌟 Key Features
- **👀 Observation Engine**: Captures screen clicks, keystrokes, and visual context.
- **🗣️ Voice-to-Intent**: Uses **Local Whisper** (Privacy-First) for offline speech transcription.
- **🧠 Local Intelligence**: Integrates with **Ollama** to synthesize skills without cloud dependencies.
- **⚡ Native Performance**: Built with **Tauri v2** (Rust + React) for a blazer-fast experience.

### 📦 Installation
We provide two ways to run Skill-E:

1.  **Installer (Recommended)**: `Skill-E_0.1.0_x64-setup.exe`
    *   Standard Windows installation.
2.  **Portable**: `Skill-E-Portable.exe`
    *   Single-file standalone executable. No installation required.

### 🛠️ Built With
- **Tauri v2** (Rust)
- **React + Tailwind**
- **Whisper & Ollama**
